# PDF-Viewer-Chrome-Extension
Extension to open and navigate in PDF file


This CHrome extension allows you to navigate through sentences using keystrokes (Tab/ Shift+Tab) inside of PDF file.

How to install and use:

1: Open you chrome browser
2: In search field, enter: chrome://extensions and press enter
3. Toggle the Developer mode on top right corner
4. After toggling, click Load unpacked on top left corner to load the extension inside the browser(select the folder of extension.
5. After successfully loading the extension, you can open your PDF files in chrome browser and use the extension to navigate and highlight sentences you're currently on.
